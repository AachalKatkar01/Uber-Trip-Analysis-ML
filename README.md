# Uber Trip Analysis using Machine Learning

## 📌 Project Overview
This project analyzes Uber trip data and builds a predictive model using Linear Regression.

## 📊 Exploratory Data Analysis
- Trips Over Time (Line Graph)
- Model Evaluation Metrics

## 🤖 Machine Learning Model
- Algorithm: Linear Regression
- Train-Test Split: 80-20
- Evaluation:
    - Mean Squared Error (MSE) 15441903660.994251
    - R2 Score  0.06790221186311607

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

This project demonstrates practical implementation of regression modeling.
